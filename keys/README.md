# MIFARE Classic Key Material

This directory is intended for templates and local authorized key files.

## Important

Do not commit real MIFARE Classic keys.

Real key files should remain local and are ignored by `.gitignore`.

Allowed in Git:

- `keys/README.md`
- `keys/templates/mfc_keys_template.json`

Ignored by Git:

- real key databases
- recovered key files
- lab-specific key material
